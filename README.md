# Typst šablóna pre VIB a VIS (FIIT STU)

Tento repozitár obsahuje šablónu pre sadzbu dokumentov v systéme [Typst](https://typst.app/), určenú pre predmety **VIB** a **VIS** na Fakulte informatiky a informačných technológií STU v Bratislave.

## Obsah repozitára

*   `main.typ`: Hlavný súbor, v ktorom sa píše obsah dokumentu.
*   `template.typ`: Súbor definujúci vzhľad a formátovanie šablóny.
*   `mybib.bib`: Súbor s bibliografickými citáciami (BibTeX formát).


## Použitie
### Webová aplikácia Typst
1. Navštívte [webovú aplikáciu Typst](https://typst.app/).
2. Vytvorte nový dokument a nahrajte súbory `main.typ`,
    `template.typ` a `mybib.bib` do projektu.

### Lokálne použitie (VSCode s Typst rozšírením)
1. Nainštalujte si [Typst rozšírenie pre VSCode](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist).
2. Vytvorte nový projekt a skopírujte/naklonujte doň súbory `main.typ`,
    `template.typ` a `mybib.bib`.
